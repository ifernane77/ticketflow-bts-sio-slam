# Fiche environnement technique — TicketFlow

## Poste de développement

- Système d’exploitation : Windows 10 / Windows 11
- Navigateur utilisé : Chrome / Firefox
- Éditeur de code : Visual Studio Code
- Serveur local : Laragon / XAMPP
- Base de données : MySQL / MariaDB
- Outil de gestion BDD : phpMyAdmin / Adminer
- Versionnage : Git
- Dépôt distant : GitHub

## Organisation du projet

Le projet est organisé en plusieurs dossiers :

- `docs/` : documentation du projet
- `src/` : code source de l’application
- `database/` : scripts SQL et modèle de données
- `tests/` : scénarios de tests
- `captures/` : preuves visuelles pour le portfolio

## Justification

J’ai préparé cet environnement afin de développer l’application TicketFlow dans de bonnes conditions, de conserver une organisation claire et de faciliter le suivi du projet avec Git.
