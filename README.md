# TicketFlow

## Présentation du projet

TicketFlow est une application web de gestion de tickets informatiques destinée à une PME fictive appelée NovaTech Services.

L’objectif est de centraliser les demandes d’assistance informatique afin d’éviter les pertes d’informations, de mieux prioriser les interventions et de permettre aux utilisateurs de suivre l’avancement de leurs demandes.

## Contexte

NovaTech Services compte environ 80 salariés. Les demandes informatiques sont actuellement transmises par mail, téléphone ou oralement. Cette organisation entraîne des oublis, un manque de suivi et une difficulté à prioriser les incidents.

## Objectif de l’application

L’application permettra :
- à un salarié de créer un ticket ;
- à un technicien de prendre en charge un ticket ;
- à un administrateur d’attribuer les tickets ;
- de suivre les statuts des demandes ;
- de conserver un historique des échanges.

## Technologies prévues

- JavaScript
